# AI Cold Email Prompts

**Product Name:** 10 ChatGPT Prompts for Killer Cold Emails

This project provides a simple landing page for a digital product: a curated pack of 10 high-quality, pre-tested AI prompts designed to generate effective cold emails using ChatGPT. 

**Features:**
- Concise and compelling landing page.
- Clear value proposition.
- Button for purchasing (Stripe integration would be added here).

**Objective:** Help small business owners, marketers, and sales professionals quickly create high-converting cold emails without experimentation.
